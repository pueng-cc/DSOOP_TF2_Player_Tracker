Searches for a player in the Container.

Details

This methode will search for a player in the Container using their name. As there can only be one of each player name it will return a single name if it exists. It will return NULL other wise as an indication that that player does not exist.

Parameters
input- string - This is the player name to search for in the Container.
Returns
Player* - Will return a pointer to the object if it exists. Otherwise will return NULL.
