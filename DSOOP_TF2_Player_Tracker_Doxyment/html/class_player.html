bool Player::MustBeDemoman ( )

Checks whether the player's only available class is Demoman.

Details

If a player's name contains "Smith", "Brown", "Wilson", "Robertson", or "Thomson" (case sensitive), the player's only valid class is Demoman.

This is different from other class validations; instead of forbidding a player to be one of the class, this validation is to forbid player to be any other class.

Returns
bool - Whether the player has to be a Demoman.

bool Player::ValidateHeavy ( )

Validates Heavy class.

Details

A Heavy cannot have "smart" within any of its field (non-case sensitive). A Heavy's melee weapon can only be "fist", "fists", or "sandvich" (non-case sensitive).

Returns
bool - Whether the player is valid as a Heavy.
