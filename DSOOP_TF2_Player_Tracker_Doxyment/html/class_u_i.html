void UI::SearchByNameUI ( )
private

Search by Name UI.

Details

This UI will prompt user to enter name of a player. It will then call Container::GetPlayerInfo() to retrieve info of the player with matching name. If such player exists, it'll call DisplayOnePlayerUI() to display the info of the player.
